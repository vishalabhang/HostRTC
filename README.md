# HostRTC
Peer To Peer Media Exchange Through WEBRTC


## Install dependencies
npm install

## Run Server
node app.js



## Access webpage 
Go to chrome browser\
ip :8080


### Note:
This Project Contains simple demo only 2 peer can exchange data. \
Further Enhancement coming soon.
